#  Artha AI: AI Based Finance Management System

Artha AI is an intelligent personal finance management system developed at **Kathmandu University, Department of Computer Science and Engineering**. It provides **data-driven financial assistance** through integrated income and expense tracking, budget planning, goal management, and predictive analytics. The system transforms raw financial data into actionable insights, enabling users to make proactive financial decisions.

---

##  Features

-  **Income & Expense Tracking**  
  Add, edit, and categorize transactions with AI-based automation.

-  **Budgeting & Forecasting**  
  Generate adaptive budgets and predict future cashflow.

-  **Goal Management**  
  Track financial goals, milestones, and receive intelligent alerts.

-  **AI Insights**  
  Personalized recommendations, predictive analytics, and “what-if” scenario analysis.

-  **Receipt Scanning**  
  Automatic transaction extraction from uploaded receipts.

-  **Chat-based Assistance**  
  Interactive chatbot for financial queries and guidance.

-  **Visual Reports**  
  Charts and dashboards to understand spending patterns and trends.

---

##  Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Django templating  
- **Backend:** Python, Django framework  
- **Database:** MySQL (secure storage of transactions, budgets, goals)  
- **AI/ML:** NumPy, Scikit-learn (predictive analytics, categorization)  
- **APIs:** OAuth for authentication, external financial advisory integration  
- **Version Control:** Git/GitHub for collaborative development

## Usage

 -Explore dashboards for trends, reports, and recommendations.
  
  ## System Design
  Artha AI follows a layered architecture:
  
 - Presentation Layer: Responsive web interface with dashboards and secure authentication.
  
 - Application Layer: Core services for transactions, budgeting, insights, goals, and reporting.
  
 - AI/ML Layer: Predictive analytics, categorization, and personalized recommendations.
  
 - Database Layer: MySQL for secure storage of user data, transactions, and financial records.
  
  ## Example Workflow
  -User uploads a receipt → AI extracts transaction details.
  
  -System categorizes expense → Updates budget and spending trends.
  
  -Predictive analytics → Alerts user about overspending risk.
  
  -Chatbot → Provides personalized advice on saving strategies.
